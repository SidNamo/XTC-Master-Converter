        XTC Master Converter
==================================================

1. 프로그램 소개
   XTC 마스터 변환기는 다양한 문서 및 이미지 파일을 전자책 단말기 전용 포맷인 
   .xtc 형식으로 일괄 변환해주는 강력한 GUI 도구입니다. 
   핵심 변환 로직은 오픈소스 프로젝트인 'epub-to-xtc-converter' 엔진을 활용합니다.

2. 주요 기능
   - 다양한 포맷 지원: TXT, PDF, EPUB 및 주요 이미지(PNG, JPG, GIF)
   - 일괄 변환: 폴더 내 모든 파일 및 하위 폴더 자동 탐색
   - ZIP 지원: 압축 파일 내부 파일을 바로 탐색하여 변환
   - 이미지 병합: 폴더/ZIP 내 이미지를 하나의 .xtc 파일로 생성
   - 실시간 모니터링: 전체 진행률 및 개별 파일 페이지 단위 상태 표시
   - 커스터마이징: 기기 해상도, 폰트(TTF/OTF), 여백, 줄 간격 설정 가능
   - 파일명 포맷: {name}, {ext}, 날짜 태그를 활용한 자유로운 이름 지정

3. 기술 스택
   - 프레임워크: Electron
   - 언어: JavaScript (Node.js)
   - 핵심 엔진: epub-to-xtc-converter
   - 주요 라이브러리: adm-zip, pdf-to-png-converter, iconv-lite, jschardet

4. 사용 방법
   1) 원본 폴더와 변환 결과 저장 폴더를 선택합니다.
   2) 사용할 폰트 파일과 기기에 맞는 해상도/여백 설정을 완료합니다.
   3) '변환 시작' 버튼을 클릭합니다.
   4) 하단 변환 이력 테이블에서 상세 상태를 확인합니다.
   5) 테이블 우클릭 시 전체 로그를 클립보드에 복사할 수 있습니다.

5. 시작하기 (개발자용)
   - 의존성 설치: npm install
   - 앱 실행: npm start
   - 빌드: npm run build
